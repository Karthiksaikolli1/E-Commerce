# Project Name
E-Commerce
## Description
This project is a Spring Boot application that manages users and products using REST APIs.

## Features
- User registration
- Login API
- CRUD operations
- Database integration

## Tech Stack
- Java
- Spring Boot
- MySQL
- Maven

## How to Run
1. Clone repo
2. Run mvn spring-boot:run
3. Open localhost:2027
